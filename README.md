# 📝 日记本

一个简约美观的个人日记/记事本网页应用，无需服务器，数据存储在本地浏览器。

## 功能

- ✅ 新建、编辑、删除记录（增删改查）
- 🔍 关键词搜索（标题 + 内容 + 标签）
- 📅 时间筛选（今天 / 本周 / 本月 / 全部）
- 😊 心情标签选择
- 🏷️ 自定义标签
- 📱 完全适配手机和电脑
- 💾 数据自动保存到浏览器本地（localStorage）

## 使用方式

### 方式一：GitHub Pages（推荐）

1. 将本仓库 fork 或上传到你自己的 GitHub
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，根目录 `/`
4. 保存后即可通过 `https://<你的用户名>.github.io/<仓库名>/` 访问

### 方式二：直接打开

下载 `index.html`，用浏览器直接打开即可使用。

## 注意

- 数据存储在当前设备的浏览器 localStorage 中
- 换设备或清空浏览器数据会导致记录丢失
- 建议定期导出备份（后续可扩展此功能）

## 技术栈

纯原生 HTML + CSS + JavaScript，零依赖，无需构建。
